# AppKit Coordination Surface

DTO-only TRINITY coordination projection surface for product-facing reads and
operator controls.

The package exposes refs for coordination runs, router decisions, role
selection, provider pools, verifier state, turn timelines, memory/context
budgets, handoffs, trace refs, and replay bundles. It does not project raw
prompts, provider payloads, model outputs, credential material, memory bodies,
or lower runtime state.
